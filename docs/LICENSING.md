# 授权方式

GLua 采用“免费非商业授权 + 付费商业授权”的双许可证模式。

## 非商业用途

个人学习、教育、研究、公益项目和其他非商业用途可以按照 [PolyForm Noncommercial License 1.0.0](https://polyformproject.org/licenses/noncommercial/1.0.0) 免费使用、修改和分发项目自有代码。

使用者必须保留许可证和 Required Notice，并继续遵守仓库内第三方组件各自的许可证。

## 商业用途

把 GLua 用于收费产品、商业服务、企业内部生产、客户交付、咨询、托管、商业基础设施或其他直接与间接商业活动之前，必须向 `zing` 取得单独的书面付费商业授权。

商业授权费用和范围按实际使用方式协商。咨询入口见 [ZingYao GitHub 主页](https://github.com/ZingYao)。不要在公开 Issue 中发布敏感商业资料。

## 许可证边界

- 根许可证只覆盖 GLua 项目拥有版权的内容。
- `third_party/` 和带独立许可证的组件继续采用各自的上游许可证。
- PolyForm Noncommercial 不是 OSI 开源许可证；公开源代码不等于允许免费商用。
- 发生冲突时，以仓库根目录 `LICENSE` 和签署的商业授权协议为准。

完整中文说明见仓库根目录的 `COMMERCIAL_LICENSE.md`，具有法律效力的非商业许可原文见根目录 `LICENSE`。
