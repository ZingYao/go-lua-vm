# 更新日志

## 1.0.0

### 中文

- 发布 VS Code glua 语言服务扩展。
- 支持 Lua/glua 语法高亮、扩展语法诊断、补全、悬停文档、跳转定义和格式化。
- 支持自定义内置函数文档 JSON，并提供中文和英文扩展描述。
- 支持连接正在运行的 GLua DAP server 进行 attach 调试。

### English

- Released the VS Code glua language server extension.
- Added Lua/glua syntax highlighting, extended syntax diagnostics, completion, hover documentation, go to definition, and formatting.
- Added custom builtin function documentation JSON support with Chinese and English extension descriptions.
- Added attach debugging for running GLua DAP servers.
